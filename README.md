# Music Library System

This project implements Object-Oriented Programming (OOP) in Python to manage a music library. 

## Features
* **Song Instantiation:** Creates individual song objects with `name`, `artist`, and `genre` attributes.
* **Global Tracking:** Uses Python class attributes and `@classmethod` decorators to track global metrics across all song instances.
* **Analytics:** Automatically calculates total song counts, unique artist lists, unique genre lists, and dictionary-based frequency counts for both artists and genres.
